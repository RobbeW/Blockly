Blockly Formularium

Doel

Denk telkens in vier vragen: Welke stappen moeten na elkaar?, Wanneer moet ik kiezen?, Wanneer moet ik herhalen zolang iets waar is?, Wanneer moet ik een vast aantal keer herhalen?

Vier programmeerconcepten

  • Eenvoudige sequentie: losse instructies staan in de juiste volgorde.
  • Selectie: het programma kiest met if, else if of else.
  • Voorwaardelijke herhaling: het programma herhaalt zolang een voorwaarde waar is, bijvoorbeeld while (notDone()).
  • Begrensde herhaling: het programma herhaalt een vooraf bepaald aantal keer, bijvoorbeeld met repeat of for.

Maze: denkstrategie

  • Eenvoudige sequentie: zet stappen in de juiste volgorde.
  • Selectie: gebruik als wanneer je moet kiezen.
  • Begrensde herhaling: herhaal een stap of patroon een vast aantal keer.
  • Voorwaardelijke herhaling: herhaal zolang je nog niet bij het doel bent.
  • Sensoren: controleer pad voor mij?, pad links? of pad rechts?.
zolang niet klaar: als pad voor mij: stap vooruit anders: draai links

Maze: dezelfde gedachte in code

while (notDone()) { if (isPathAhead()) { moveForward(); } else { turnLeft(); } }

Let op de haakjes bij functies: notDone() en isPathAhead().

Turtle: beweging

  • Begrensde herhaling: herhaal een bouwsteen een vast aantal keer wanneer je een vorm of patroon tekent.
  • moveForward(100): ga vooruit.
  • moveBackward(50): ga achteruit.
  • turnLeft(90) en turnRight(90): draai in graden.
  • goTo(x, y): spring naar een coordinaat. Het midden is (0, 0).
  • penUp() en penDown(): verplaatsen zonder of met tekenen.

Turtle: tekenen

  • setPenColour('#5200FF'): kies een kleur.
  • setPenColour(82, 0, 255): kies een RGB-kleur.
  • setLineWidth(4): pas de lijndikte aan.
  • writeText('Hallo'): schrijf tekst op de huidige plaats.
  • fitToDrawing(): pas de tekening in beeld.

Snelle foutcheck voor je op Uitvoeren klikt

  • Heb je bij code functies met haakjes geschreven, zoals moveForward(100)?
  • Verandert er iets in je while-lus, zodat de lus ooit stopt?
  • Test eerst een kleine lijn, daarna een vorm, daarna pas een patroon.
  • Staat de pen omhoog wanneer je alleen wil verplaatsen?
  • Bij Maze: verlaag de snelheid en kijk waar je algoritme een verkeerde keuze maakt.
  • Bij Turtle: controleer hoeken. Een vierkant gebruikt meestal 90 graden, een gelijkzijdige driehoek 120 graden draaien.

Kleurkiezer

Werkt in Turtle-modus. In code kan je dezelfde kleur gebruiken met setPenColour('#5200FF').
